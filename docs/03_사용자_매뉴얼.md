# 사용자 매뉴얼 — 올댓스파인 스마트 업무관리 시스템 (SWMS)

| 항목 | 내용 |
|------|------|
| **문서 버전** | v1.0 (시스템 v1.0.0 기준) |
| **작성일** | 2026-06-05 |
| **대상** | 전 직원 · 회계담당 · 대표(관리자) |

> 본 매뉴얼은 **실제 업무 시나리오** 순서로 구성했습니다. 처음 쓰실 땐 0장 → 본인 역할 장(1~3장)을 먼저 보세요.

---

## 0. 시작하기

### 0-1. 로그인
1. 회사에서 받은 **이메일·임시 비밀번호**로 로그인 화면에서 로그인합니다. (계정은 대표가 생성하며, 셀프 가입은 없습니다.)
2. 최초 로그인 시 **비밀번호 변경**을 권장합니다.
3. 로그인 화면 배경은 회사 브랜드 이미지(척추+AI), 가운데 로그인 카드가 표시됩니다.

### 0-2. 화면 구성
- **왼쪽 사이드바**: 모듈 메뉴(대시보드/근태/업무보고/나의 업무/프로젝트/예산🔒/게시판/자료공유/캘린더/전자결재/관리자🔒). 권한에 따라 일부 메뉴는 보이지 않습니다.
- **상단 헤더**: 가운데 시스템명, 오른쪽 🔔 알림 벨.
- **로그인하면 대시보드(상황판)** 가 먼저 뜹니다.

### 0-3. 알림
- 결재 요청/결과, 댓글, 필독 공지 등은 🔔 벨에 쌓입니다. 클릭해 해당 화면으로 이동합니다.

## 1. 직원(일반) 시나리오

### 1-1. 출근하기 (근태)
1. 회사 도착 후 사이드바 **근태관리** → **출근** 버튼.
2. **사무실 QR**을 스캔(모바일)하거나 QR 아래 **4자리 코드**를 입력(PC).
3. 서버가 ① 접속 IP가 회사 IP 대역인지 ② QR 토큰이 유효한지 확인 후 출근 처리합니다.
   - 09:00 이후면 자동으로 **지각** 처리됩니다.
4. 퇴근 시에도 동일하게 **퇴근** 처리(18:00 이전이면 조기퇴근).

> ⚠️ 회사 IP가 아니거나 QR이 틀리면 출근이 거부됩니다(원격 우회 차단). 누락 시 **정정 요청** → 대표 승인.

### 1-2. 오늘 일일보고 작성
1. **업무보고** → **새 보고 작성** → 유형 **일일**.
2. 그날 한 업무를 **여러 건** 추가할 수 있습니다. 각 업무에 **제목·연관 프로젝트·진행상태·내용(3~4줄)·소요시간** 입력.
3. 저장하면 대시보드의 "오늘 일일보고"가 ✔ 처리됩니다.

### 1-3. 나의 업무(목표) 관리
1. **나의 업무** → 주간/월간 목표를 작성(직원 제안 또는 대표 지시).
2. 측정 가능한 **목표지표·달성기준**을 적고, 진행하며 **진행률**을 갱신합니다.
3. 업무보고를 목표에 연결하면 진행 현황이 자동 반영됩니다.

### 1-4. 휴가 신청
1. **근태관리** → 휴가 신청(연차/반차/병가/경조사 등), 기간·사유 입력 → 제출.
2. 대표가 승인하면 **연차 잔여**가 차감되고, **캘린더 부재 현황**에 반영됩니다.
   - (전자결재의 "휴가 신청"으로도 가능 — 승인 시 근태에 자동 반영)

### 1-5. 전자결재 상신
1. **전자결재** → **새 상신** → 종류(휴가/지출/구매/일반기안) 선택 → 양식 입력 → 상신.
2. 대표에게 알림이 가고, 승인/반려되면 내게 알림이 옵니다. 대기 중이면 **회수** 가능.
3. 처리된 문서는 **정식 결재문서(A4)** 로 표시되며 **「인쇄/PDF」** 로 출력할 수 있습니다.

### 1-6. 게시판·자료·캘린더
- **게시판**: 공지 확인(필독은 알림), 자유글·댓글·이모지, 익명 건의(대표만 열람).
- **자료공유**: 사규·양식 다운로드, 파일/외부링크 업로드.
- **캘린더**: 내 일정·회사 일정·휴가·마일스톤·공휴일을 한 화면에서 확인, 개인/회사 일정 등록.

### 1-7. 내 대시보드
로그인 직후 화면에서 **오늘 출퇴근·오늘 보고·진행 중 나의 업무·내 결재 상태**와 **해야 할 일(To-do)** 을 바로 확인합니다.

## 2. 회계담당 시나리오 (예산 🔒)

> 예산 메뉴는 **회계담당·대표만** 보입니다.

### 2-1. 초기 1회 — 계정 등록
1. **예산** → **계정 관리** → 통장/법인카드/현금을 등록하고 **기초 잔액**(시작 시점 잔액)을 입력.

### 2-2. 매일 — 입출금 기록
1. **예산** 메인의 **일별 현황**에서 날짜를 고르고 **거래 추가**.
2. 구분(입금/출금), 계정, 금액, **적요(필수)**, 거래처 입력. 금액은 천 단위 콤마가 자동 표시됩니다.
3. **카드 결제는 카드 계정에 '출금'** 으로 입력 → 카드 사용으로 집계되고 **통장 출금과는 분리**됩니다.
   - 실제 카드대금이 통장에서 빠질 때, 그때 **통장 '출금'** 으로 따로 입력합니다(이중 계상 방지).
4. 계정별 잔액이 자동 추정되고, 월별/연도별 **통계·그래프**로 확인합니다.

### 2-3. 매출 목표 추적
1. **예산** → **매출 목표** → 연도 선택 → 매출 항목과 **연간 목표(필수)·월간 목표(선택)** 등록.
2. 입금 거래에 **매출 항목**을 지정하면, 예산 메인 상단 **매출 현황**에 연간 달성률·항목별 비중·**월별 추이 그래프**가 표시됩니다.

## 3. 대표(관리자) 시나리오 🔒

### 3-1. 전사 상황판(대시보드)
로그인하면 **전사 현황 상황판**:
- **KPI**: 오늘 출근(N/M)·일일보고 제출·결재 대기·주의 프로젝트.
- **처리 대기함**: 휴가 승인·근태 정정·전자결재·나의 업무 승인 건수를 한곳에서 → 클릭해 처리.
- **위젯**: 오늘 근무 현황(미출근·휴가 명단), 프로젝트 신호등(세부업무 펼침), 보고 미제출자, 다가오는 일정, 예산 현황, 최근 공지.

### 3-2. 직원 계정 관리
1. **관리자** → **직원 관리** → 직원 추가(이름·이메일·역할·입사일·현재 사용 연차) → 임시 비밀번호 발급.
2. 역할(직원/회계/부서장/대표) 부여, 재직상태 변경, 퇴사 처리(기록 보존).

### 3-3. 승인 처리
- **휴가/근태 정정**: 근태관리에서 승인 대기 목록 확인 → 승인/반려.
- **전자결재**: 결재함에서 승인(서명 자동 표시)/반려(사유 필수). 휴가 결재 승인 시 근태·캘린더 자동 반영.
- **나의 업무**: 직원 목표 승인.

### 3-4. 운영 설정
**관리자** 메뉴에서:
- **근무 규칙**: 출퇴근 기준시각·지각 유예·보고 마감.
- **허용 IP 대역**: 출퇴근 가능한 회사 공인 IP(배포 후 **반드시 실제 공인 IP로 교체**).
- **보고 템플릿**: 일/주/월 보고 항목 편집.
- **프로젝트 태그**: 보고·미션에 연결할 프로젝트 등록.
- **사무실 QR**: 출퇴근 인증 QR 발급(포스터 부착).
- **전자결재 직인·서명**: 회사 직인·본인 서명 이미지 업로드(투명 PNG 권장) → 결재문서에 표시.

### 3-5. 데이터 백업/복원
**관리자 → 데이터 백업**:
- **지금 백업**: 전체 데이터를 스냅샷으로 클라우드에 저장. 목록에서 **다운로드**(이중 보관 권장).
- **자동 백업**: 매일 새벽(배포 환경) + **로그인 시 자동 다운로드 스위치**(기본 꺼짐, 켜면 로그인마다 내 PC로 저장).
- **복원**: 백업을 골라 "복원" 입력 후 실행. 백업의 행을 되살리거나 갱신(기존 행 삭제 없음).

## 4. 자주 묻는 질문 (FAQ)

| 질문 | 답 |
|------|-----|
| 출근 버튼을 눌러도 안 돼요 | 회사 IP가 아니거나 QR/코드가 틀린 경우입니다. 사무실 네트워크에서, 올바른 QR로 시도하세요. |
| 비밀번호를 잊었어요 | 대표(관리자)에게 비밀번호 초기화를 요청하세요. |
| 예산 메뉴가 안 보여요 | 회계담당·대표만 접근 가능합니다. |
| 결재문서를 PDF로 받고 싶어요 | 결재 상세에서 **「인쇄/PDF」** → 인쇄창에서 "PDF로 저장". |
| 금액에 콤마가 자동으로 들어가나요 | 네, 예산 입력란은 천 단위 콤마가 자동 표시됩니다. |
| 백업 파일은 어디에 저장되나요 | 클라우드(Storage) + 다운로드 시 내 PC. 자동 다운로드는 대시보드 스위치로 켭니다. |
| 휴가가 통계에서 빠지나요 | 승인된 휴가일은 결근/지각 통계에서 자동 제외됩니다. |

## 5. 용어

- **상황판(대시보드)**: 한 화면에서 현황을 보는 메인 화면.
- **신호등**: 프로젝트 상태(정상🟢/지연🟡/위험🔴).
- **적요**: 거래 내용 설명(예: "1월 임대료").
- **결재함**: 내가 승인해야 할 문서 목록.
