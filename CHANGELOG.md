## [1.0.0] - 2026-06-05

올댓스파인 스마트 업무관리 시스템 **첫 정식 릴리스**. Phase 1~3 전 모듈 구현 완료.

### Added — 핵심(Phase 1)
- **인사정보**: 관리자 직원 계정 생성(service_role), 역할(직원/회계/부서장/대표/키오스크), 입사일 기준 **연차 자동 계산**, 재직상태.
- **근태관리**: 출퇴근 체크 — **회사 IP(CIDR) + 사무실 QR** 이중 위치 인증, **1시간 단위 회전 QR**(TOTP식, 전용 키오스크 계정), 4자리 수동코드, 서버 시각 기준 기록, 지각/조기퇴근 자동 판정, 휴가 신청/승인, 근태 정정, **일별·직원별·월별·연간 통계**, 공휴일 자동 인식.
- **업무보고**: 일/주/월 동적 템플릿(관리자 커스터마이즈), 일자별 다중 업무 기록(3~4줄), 프로젝트 연계, 관리자 코멘트.
- **나의 업무(목표관리)**: 주/월/일/기한없음 목표, 진행률, 프로젝트 연계, 승인 흐름.
- **통합 대시보드 · 인앱 알림**.

### Added — 모니터링(Phase 2)
- **프로젝트**: 블로그형 게시판(리치 에디터·이미지·파일), 세부 업무 체크리스트(게시글 연결), 마일스톤, 신호등 상태.
- **예산(라이트)**: 통장/법인카드/현금 계정·기초잔액, **일별 입출금 원장**(적요 필수), 계정별 잔액 자동 추정, **카드 사용=출금과 분리**(이중 계상 방지), 월별·연도별 통계(차트), **매출 목표 추적**(연간/월간 목표 대비 달성률·항목별 비중·월별 추이 그래프). 회계담당·대표만 접근(RLS SELECT까지 차단).

### Added — 보조(Phase 3)
- **게시판**: 공지/자유/익명건의, 댓글·대댓글·이모지, 필독·읽음확인, **로그인 팝업 공지**(오늘/다시 안보기).
- **자료공유**: 폴더, 파일 업로드 + 외부 링크, 전체/지정 공개.
- **캘린더**: 휴가·마일스톤·프로젝트 기한·미션 마감·공휴일 **자동 합성 월 그리드**, 부재 현황, 일정 등록.
- **전자결재**: 휴가/지출/구매/기안 신청→승인/반려(사유)/회수, 코멘트, **A4 세로 결재문서**(결재란·문서번호·회사명) + **회사 직인·결재자 서명** 이미지, 브라우저 인쇄(PDF). 휴가 승인 시 근태 자동 반영.

### Added — 운영·디자인
- **역할별 종합 대시보드(상황판)**: 대표=KPI(출근/보고/결재대기/주의 프로젝트)+처리 대기함+위젯, 직원=내 오늘+할 일+미션 진행률, 회계=예산 위젯. 프로젝트 신호등 세부업무 펼침.
- **데이터 백업/복원 시스템**: 전 31개 테이블 + 계정 스냅샷 JSON, 비공개 Storage 보관, **매일 자동(Vercel Cron)** + 수동 + **로그인 시 로컬 자동 다운로드(관리자 스위치, 기본 OFF)**, 의존성 순서 merge 복원(관리자 전용·확인 절차).
- **디자인**: 브랜드 틸-블루 테마, **Pretendard** 기본 폰트, 척추+AI 네트워크 모티프(로그인 풀스크린 배경·대시보드 히어로·빈 상태 일러스트), 중앙 정렬 헤더 타이틀.
- 금액 입력 천 단위 콤마, OG 이미지/메타데이터.

### Security
- 전 테이블 **RLS(Row Level Security)** 적용, 권한 헬퍼(`is_admin`, `is_accountant_or_admin`).
- 신뢰 로직(IP/QR 검증, 권한 판정)은 서버에서만. 시간은 서버 시각(KST).
- `service_role` 키 서버 전용. 예산/인사 데이터 이중 방어.

### 개발 단계 요약
`STEP0 기반` → `인사` → `근태` → `업무보고` → `나의 업무` → `프로젝트` → `예산(+매출목표/통계)` → `Phase3 4종` → `대시보드 상황판` → `백업` → `디자인·마무리`.
